# au-eastcoast-otp-network
OpenTripPlanner 2.6 multimodal routing network for Australia's east coast (Melbourne–Canberra–Sydney–Newcastle). Includes GTFS feeds for rail, bus, tram, ferry, domestic flights, and private intercity coaches, with GTFS Fares V1 across all modes. Built for the ARC Discovery Project on sustainable city size and high-speed rail demand modelling.
